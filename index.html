license

Create LICENSEs from the command-line

View the Project on GitHub

Install 

go get github.com/nishanths/license

More install options

Usage

Generate license

To generate a license, simply run license followed by the license name. The following command generates the MIT license:

license mit

Create license file

Use the -o option to save the license to a file. The following command creates the file LICENSE.txt with the contents of the ISC license:

license -o LICENSE.txt isc

Customize name and year

The program makes a clever guess for the name and year to use on the license, but you can also explicity specify them:

license -n Alice -y 2013 isc

Additional options are described in the README.
